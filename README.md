## 📝 Description

WeeklyExpenditureTracker is a desktop Java Swing application designed to simplify personal budgeting by tracking weekly expenses. It addresses the need for a straightforward, offline, and immediate way to log daily spending across different categories without the overhead of complex, web-based tools. The application operates entirely locally, making it a secure choice for quick financial entry.

Users can input their expenditures day by day, specifying categories for each entry. The desktop GUI application processes these inputs, validates the entered data for correctness, and dynamically calculates totals while grouping expenditures by category. This structure provides users with an instant, clear overview of their weekly spending habits.

## ✨ Key Features

- **📝 Daily Expense Entry** — Input and record expenditures systematically categorized by the day of the week.
- **🗂️ Category-Based Grouping** — Organize and view expenses grouped by specific spending categories for structured tracking.
- **🧮 Automated Total Calculation** — Automatically calculate and display total expenditures for quick financial summaries.
- **✔️ Built-in Input Validation** — Validate user inputs during entry to prevent errors and maintain accurate budget data.

## 🎯 Use Cases

- An individual looking for a lightweight, offline desktop tool to log and track weekly personal spending.
- A budget-conscious student wanting to categorize expenses and view dynamic totals without complex software.
